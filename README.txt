# AI Бот Серфер

Приложение для автоматического веб-серфинга с использованием нескольких AI ботов, способных имитировать человеческое поведение.

## Возможности

- Управление несколькими ботами одновременно
- Создание сложных задач на естественном языке
- Загрузка изображений для анализа ботами
- Полная история выполнения задач
- Имитация человеческого поведения (движения мыши, скроллы, случайные задержки)

## Установка

1. Убедитесь, что у вас установлен Node.js (версия 18+)
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-repo/ai-bot-surfer.git
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```

## Запуск

Для разработки:
```bash
npm run dev
```

Для production:
```bash
npm start
```

Приложение будет доступно по адресу: [http://localhost:8000](http://localhost:8000)

## Конфигурация

Основные настройки можно изменить в файле `config.json`:

```json
{
  "bots": {
    "maxCount": 10,
    "defaultTimeout": 30000
  },
  "google": {
    "searchUrl": "https://www.google.com/search?q="
  }
}
```

## Использование

1. На главной странице выберите количество ботов
2. Перейдите в "Конструктор задач" и опишите задачу на естественном языке
3. При необходимости загрузите изображения
4. Запустите выполнение задачи
5. Отслеживайте прогресс в разделе "Логи"
